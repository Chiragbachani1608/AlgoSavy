Implementing Searching Algorithms in Python

In Python, you can implement searching algorithms to find elements in arrays or lists. Here's an example of Binary Search:

def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = left + (right - left) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

You can create functions for other searching algorithms in a similar way.
